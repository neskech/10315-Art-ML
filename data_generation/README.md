# Script for downloading data:

Modify the following field in download_data.py with the desired file path tag.
For example, when viewing the URL for a ZIP file in Google Drive in a new tab, for "https://drive.google.com/file/d/samplepathname/view", it would be:

```python
FILE_ID = "samplepathname"
```

Then, run the following command:
```bash
python3 download_data.py 
```
